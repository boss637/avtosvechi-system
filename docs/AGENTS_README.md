## 🏗️ АРХИТЕКТУРА АГЕНТОВ

### 1. 👁️ **WATCHER AGENT** — Агент Наблюдения  
**Назначение:** Мониторинг системы в реальном времени  
**Порт:** 9090  
**Директория:** `watcher-agent/`  
**Функции:**  
- 📊 Отслеживание метрик системы  
- 🔍 Обнаружение аномалий  
- 📝 Сбор логов и диагностика  
- 🚨 Генерация алертов  
**Технологии:** Prometheus, Grafana, custom metrics


### 2. 🤖 **EXECUTOR AGENT** — Агент Исполнитель  
**Назначение:** Автоматическое выполнение действий  
**Порт:** 8000  
**Директория:** `executor-agent/`  
**Функции:**  
- 💬 Обработка команд из Telegram  
- ⚡ Выполнение скриптов  
- 🎛️ Управление ресурсами  
- 🔄 Автоматическое восстановление  
**Технологии:** Python, Telegram Bot API, Docker API


### 3. 🧠 **SOLVER AGENT** — Агент Решатель  
**Назначение:** Анализ и решение сложных проблем  
**Порт:** 8080  
**Директория:** `solver-agent/`  
**Функции:**  
- 🧩 Анализ корреляций событий  
- 💡 Предложение решений  
- 📈 Прогнозирование проблем  
- 🎯 Оптимизация работы системы  
**Технологии:** Machine Learning, правила, алгоритмы
